# Project Razer 🎮

> **Notice:** This is a student school project created strictly for educational purposes as part of a front-end development activity. 

This project is a front-end replica inspired by a reference photo of the Razer website in Pinterest. All rights, logos, and asset designs belong to **Razer Inc.** 

---

## 📌 Project Scope & Limitations

* **Front-End Exercise:** Developed purely as a visual layout practice to match a design reference.
* **Web View Only:** This project is optimized **only for desktop web view** and is *not* built to be responsive on mobile or tablet screens.
* **Static Layout:** No backend or functional e-commerce features are implemented.

---

## 🛠️ Tech Stack

* **HTML5** - Structure and layout
* **CSS3** - Styling, themes, and positioning
